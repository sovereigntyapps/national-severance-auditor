# National Severance Auditor 🇨🇦

An AI-powered legal audit tool designed to navigate the complexities of Canadian employment law.

### Key Features
* **Jurisdictional Intelligence:** Automatically toggles between Provincial ESA and Federal Labour Code (CLC) logic.
* **Agentic Reasoning:** Uses Gemini 1.5 Pro 'High Thinking' to analyze Bardal Factors (Age, Tenure, Seniority).
* **Mathematical Precision:** Integrates Python Code Execution to ensure 100% accuracy in statutory calculations.
* **SWA Architecture:** Built with a local-first, serverless mindset for maximum privacy and performance.

### How it Works
The auditor calculates the **Statutory Floor** (what the law requires) and contrasts it with the **Common Law Ceiling** (what precedents suggest), giving users a powerful negotiation baseline.
